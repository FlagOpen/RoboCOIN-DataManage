# RoboCOIN DataManager

**版本: v1.1**

在线演示: https://flagopen.github.io/RoboCOIN-DataManager/

## RoboCOIN DataManager 是什么？

RoboCOIN DataManager 是一个用户友好的网页应用程序，帮助您轻松浏览、探索和下载 RoboCOIN 项目的数据集。无论您是研究人员、开发者还是机器人爱好者，这个工具都提供了直观的界面来发现和获取您项目所需的机器人操作数据。

### 主要功能
- **浏览 RoboCOIN 数据集**：通过视频预览探索数百个机器人操作片段
- **高级筛选**：按场景、机器人类型、动作和对象查找特定数据集
- **视频预览**：直接在浏览器中观看机器人任务的短视频片段
- **批量下载**：选择多个数据集并生成批量获取的下载命令
- **多源支持**：支持从 ModelScope 和 HuggingFace 仓库下载

> ⚠️ **重要提醒**：在使用此网页界面下载数据集之前，请先安装 [RoboCOIN 项目](https://github.com/FlagOpen/RoboCOIN)，以获得预先编写好的批量下载脚本的最佳下载体验：
> ```bash
> pip install robocoin
> ```

## 网页应用的工作原理

这个网页应用直接在您的浏览器中加载 RoboCOIN 数据集信息，并提供交互式界面来探索和选择数据集。下面是它的工作原理：

### 数据加载过程
1. **数据集发现**：应用加载所有可用 RoboCOIN 数据集的元数据
2. **视频预览**：加载短视频片段用于预览机器人任务
3. **交互式筛选**：您可以按各种标准筛选数据集，如机器人类型、场景和动作
4. **下载生成**：选定的数据集用于生成 Python 下载命令

### 浏览器要求
- **现代浏览器**：Chrome 61+、Firefox 60+、Safari 11+、Edge 61+ 或 Opera 48+
- **JavaScript**：必须启用（应用使用现代 ES6 模块）
- **网络连接**：需要加载数据集信息和视频预览

## 主要功能

### 1. 数据集探索
- **视觉预览**：观看机器人操作任务的短视频片段
- **详细信息**：查看每个数据集的完整元数据，包括机器人类型、场景、动作和对象
- **响应式设计**：在桌面、平板和手机上无缝工作

- **多个筛选类别**：按场景、机器人型号、末端执行器类型、动作类型和操作对象筛选

- **快速重置**：一键清除所有筛选器

- **视觉选择**：点击数据集卡片进行选择/取消选择

- **多源下载**：选择从 ModelScope 或 HuggingFace 下载
- **Python 命令**：获取即用的 Python 下载命令

- **选择备份**：将您的数据集选择导出为 JSON 文件

### 5. 用户友好界面
- **加载进度**：加载数据集时查看进度条
- **错误处理**：如果出现问题，会显示清晰的错误消息
- **提示通知**：对您的操作提供有帮助的反馈
- **键盘快捷键**：使用 Ctrl+F 快速搜索筛选选项

## 开始使用

2. 等待数据集加载（您会看到进度条）
3. 开始探索 RoboCOIN 数据集！

### 浏览器兼容性
- **推荐**：Chrome 61+、Firefox 60+、Safari 11+、Edge 61+
- **JavaScript**：浏览器中必须启用
- **网络**：需要加载数据集信息

## 如何使用网页应用

### 第 1 步：探索数据集
1. **浏览视频**：滚动查看数据集卡片，观看机器人任务的视频预览
2. **悬停获取信息**：悬停在任何视频卡片上查看基本信息，无需点击
3. **点击查看详情**：点击视频卡片打开包含完整数据集信息的模态框

### 第 2 步：筛选您的搜索
1. 点击顶部控制栏中的 **"Filter datasets"** 按钮
2. 使用侧边栏选择筛选类别：
   - **场景**：按环境筛选（厨房、客厅等）
   - **机器人**：按机器人型号筛选（Franka、UR5 等）
   - **末端执行器**：按夹爪类型筛选
   - **动作**：按任务类型筛选（拾取、放置等）
   - **对象**：按操作对象筛选
3. 点击筛选选项激活它们（您可以选择多个）
4. 使用搜索框（Ctrl+F）快速查找特定筛选选项
5. 点击 **"Done"** 或按 Escape 键应用筛选

### 第 3 步：按名称搜索
- 使用顶部控制栏中的搜索框按名称搜索数据集
- 搜索与您的活动筛选器结合使用，以获得精确结果
- 点击 × 按钮清除搜索

### 第 4 步：选择数据集
1. **单个选择**：点击视频卡片进行选择/取消选择（选中的卡片会显示边框和勾选标记）
2. **批量选择**：点击 **"Select all datasets"** 选择当前可见的所有数据集
3. **清除选择**：点击 **"Deselect all datasets"** 清除所有选择
4. 在顶部栏中查看选择计数器

### 第 5 步：生成下载命令
1. 您选定的数据集会出现在右侧的 **"Batch Downloader"** 面板中
2. 选择您的首选下载源：**HuggingFace** 或 **ModelScope**
3. 点击 **"Copy & Checkout"** 复制 Python 下载命令
4. 在终端中粘贴并运行命令下载数据集
5. **可选**：添加 `--target-dir /your/custom/path` 指定自定义下载位置

### 第 6 步：保存您的选择
- **导出选择**：点击 **"Export JSON"** 将当前选择保存为文件
- **导入选择**：点击 **"Import JSON"** 加载之前保存的选择
- 适合分享选择或稍后继续工作

### 使用技巧
- **视频加载**：悬停在视频上时会自动加载
- **性能**：应用使用虚拟滚动来高效处理大量数据集
- **移动端友好**：支持平板和手机的触控操作
- **错误恢复**：如果出现问题，请尝试刷新页面
